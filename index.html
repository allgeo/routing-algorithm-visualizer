General note (for group members)

Although the graph alter its orientation everytime you refresh, all the nodes, edges and weights are the same. And there is a way to make the graph static using cytoscape.

The edges are weighted and the nodes are labeled with their names. You can acesses and update each node using the cytoscape API. For example, to change the weight of the edge between node n1 and node n2, usinng the following code:

cy.$('#n1n2').data('weight', 10);

Could have a seperate input boxes for each of the edges, then update the edges using the cytoscape API once user submits the form.
