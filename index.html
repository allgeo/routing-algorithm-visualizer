Update as of March 9th, 2023 (for group members)

Whats Done
  • Uses cytoscapes.js to generate and and manupulate the graph.
  • Uses Bootstrap 5.2 to for styling and components
  • Users are able to update the graph weight by entering values into the input field, then by pressing the submit button.
  • The submitted weight is presistant since its stored in browser local storage. Meaning, refreshing the page will keep the previously entered data. As long as the browser cache isn't cleared, the previous data will be kept.
Whats Left
  • At the moment, cytoscapes stores the weight values as string (even if its a number, its represented as string). Need to implement a validation so that users can only enter numbers.
  • Whenever you refresh the page, the orientation of the graph changes (but all the nodes and edges are same). There is a way to hardcode each node to a set point using cytoscape but still need to do that
  • The "Choose a Routing Algorithm" dropdown isnt functional, since we havent inplement the algorithm for it it.
  • There is also a way to highlight nodes and edges using cytoscape, so we could use that to highlight the path that the algorithm takes.
